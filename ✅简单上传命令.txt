✅ GitHub上传 - 复制粘贴这些命令即可
══════════════════════════════════════════════════════════════════

在项目目录打开命令行（PowerShell或CMD），依次运行：

══════════════════════════════════════════════════════════════════
第1步：初始化Git
══════════════════════════════════════════════════════════════════

git init

══════════════════════════════════════════════════════════════════
第2步：设置远程仓库
══════════════════════════════════════════════════════════════════

git remote add origin https://github.com/Keria04/DashboradGrounding.git

══════════════════════════════════════════════════════════════════
第3步：配置Git LFS（处理大文件）
══════════════════════════════════════════════════════════════════

git lfs install
git lfs track "*.pt"
git add .gitattributes

══════════════════════════════════════════════════════════════════
第4步：拉取远程代码（合并现有内容）
══════════════════════════════════════════════════════════════════

git pull origin main --allow-unrelated-histories --no-edit

（如果提示没有main分支，忽略错误继续）

══════════════════════════════════════════════════════════════════
第5步：添加所有文件
══════════════════════════════════════════════════════════════════

git add .

══════════════════════════════════════════════════════════════════
第6步：提交到本地
══════════════════════════════════════════════════════════════════

git commit -m "Phase 1完成: YOLOv8s检测系统 (mAP50=51.2%)"

══════════════════════════════════════════════════════════════════
第7步：设置主分支
══════════════════════════════════════════════════════════════════

git branch -M main

══════════════════════════════════════════════════════════════════
第8步：推送到GitHub
══════════════════════════════════════════════════════════════════

git push -u origin main

输入凭据：
  Username: 你的GitHub用户名
  Password: 粘贴你的Personal Access Token（不是密码！）

══════════════════════════════════════════════════════════════════
✅ 完成！
══════════════════════════════════════════════════════════════════

访问查看：https://github.com/Keria04/DashboradGrounding

══════════════════════════════════════════════════════════════════
📌 如何获取Personal Access Token
══════════════════════════════════════════════════════════════════

1. 访问: https://github.com/settings/tokens
2. 点击: Generate new token (classic)
3. 填写:
   Note: DashboradGrounding
   Expiration: 90 days
   勾选: ✅ repo
4. Generate token
5. 复制token（只显示一次！）
6. 推送时用token作为密码

🔧 常见错误处理
══════════════════════════════════════════════════════════════════

错误: Permission denied
解决: 联系仓库所有者添加您为Collaborator

错误: Authentication failed  
解决: 使用Token而不是GitHub密码

错误: LFS not found
解决: 下载安装 https://git-lfs.github.com/

错误: File too large
解决: 确保Git LFS已正确配置
