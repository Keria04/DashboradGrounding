⚠️ 可能遇到的问题
══════════════════════════════════════════════════════════════════

问题1: Permission denied (权限拒绝)
────────────────────────────
原因: 您不是仓库的Collaborator
解决: 联系 Keria04 添加您为协作者
     Settings → Collaborators → Add people

问题2: Authentication failed
────────────────────────────
原因: Token错误或过期
解决: 重新生成Token，确保勾选repo权限

问题3: Updates were rejected
────────────────────────────
原因: 远程有新的提交
解决: git pull origin main --rebase
     git push -u origin main

问题4: LFS bandwidth limit
────────────────────────────
原因: LFS流量超限（Git LFS免费1GB/月）
解决: 等待下月重置，或升级Git LFS
